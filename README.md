## Mac VS Code 编译器中 Copilot 的快捷键 

- 接受建议：Tab
- 拒绝建议：Esc
- **触发行内Copilot：Alt/Option + \** （Coplit还没有给出建议或者建议被拒绝了，希望手工触发它提供建议）
- **打开Copilot：Ctrl + Enter （会打开一个单独的面板，展示10个建议）**
- 下一条建议：Alt/Option + ]
- 上一条建议：Alt/Option + [

## 本地环境应该有的工具

- 单元测试工程，用于各种简单的测试验证

- 错误码查询工具，用于快速查询错误码的含义
- 接口文档，用于查看接口的定义
